<h2 align="left">Local installation</h2>


<p align="left">Use this commandes below to run the program </p>

```
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ rails server 
```

Ruby version : 2.7.4
Rails version : 5.2.6
